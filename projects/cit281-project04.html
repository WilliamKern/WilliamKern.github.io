Overview

Project 04 centered around building a REST API using Node.js and Express. I created a p4-module.js file that stores and manages an array of question/answer objects, then used an Express server to provide endpoints for retrieving, adding, and deleting questions via HTTP requests. This project helped solidify my understanding of REST architecture and how servers interact with data modules.

Key Learning Outcomes

  • Designed and implemented a simple REST API
  • Created and exported multiple data-handling functions from a module file
  • Used Express routing to create GET, POST, and DELETE endpoints
  • Parsed URL parameters and request bodies with express.json()
  • Sent structured responses using res.json() and proper status codes
  • Organized routes and validated request data
  • Structured modular, testable server code
  • Separated data logic from routing concerns
  • Handled missing or invalid input appropriately
