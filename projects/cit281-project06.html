Preview of the task management application interface with CRUD functionality

Preview of the task management application interface with CRUD functionality

Overview

Project 06 focused on developing a full CRUD (Create, Read, Update, Delete) web application using Express and modular JavaScript classes. I built a Task class and a TaskList manager, then created an Express server with routes that allow tasks to be created, retrieved, updated, or deleted via API requests. I also added a front-end interface to interact with these features, using fetch() and dynamic HTML rendering.

Key Learning Outcomes

  • Created JavaScript classes to represent individual tasks and a manager class to store them
  • Built RESTful API routes using Express for GET, POST, PUT, and DELETE operations
  • Stored and updated task data in memory with proper JSON responses
  • Connected the frontend with the server using fetch() and dynamic DOM updates
  • Handled edge cases and error messaging for invalid input or task IDs
  • Organized server code and managed class-based data structures
  • Implemented asynchronous request handling
  • Designed and built a full-stack web application
