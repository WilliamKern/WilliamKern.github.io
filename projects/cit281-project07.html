Overview

Project 7 involved creating this very portfolio website using GitHub Pages. The goal was to develop a comprehensive showcase of all the projects and labs completed throughout CIT 281. This website serves as both a demonstration of web development skills and a documentation of the learning journey through the course.

Key Learning Outcomes

  • Developed a responsive portfolio website using modern HTML5 and CSS3
  • Implemented dynamic carousels for project screenshots using Swiper.js
  • Created consistent layouts and styling across multiple project pages
  • Utilized GitHub Pages for web hosting and deployment
  • Applied best practices for web accessibility and user experience
  • Organized and documented multiple projects in a cohesive manner
  • Integrated Font Awesome icons and other external resources
  • Implemented responsive navigation and modal image viewing
