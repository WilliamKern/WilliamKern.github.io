Overview

For this project, I created the "Ultimate Coin Tool" web application with two basic capabilities: calculating all possible US coin combinations given a specific target number of pennies, and calculating the value in pennies of a specific number of each of the available US coins.

Key Learning Outcomes

  • Gained experience interpreting functional descriptions and specifications
  • Used modular programming by breaking the project into manageable components
  • Wrote and executed non-web server Node.js JavaScript code using VSCode
  • Created and used code modules
  • Practiced refactoring using modern JavaScript syntax
  • Worked with JSON data structures
  • Used git version control and created standard Node .gitignore file
  • Wrote and executed web server Node JavaScript code using VSCode
  • Used Express with GET verb, routes, and query parameters
  • Loaded and served "public" files as web pages
